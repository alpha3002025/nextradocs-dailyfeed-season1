- 지금 다시 돌아보니 인스톨 스크립트도 더 단순하게 만들수 있었다. 이 프로젝트 규모에서 필요하게 될 요소들이 모두 뭔지 모르는 상황이었기에 인스톨 스크립트가 점점 복잡해진 것




애플리케이션의 코드의 경우 dailyfeed-member 의 코드만 복잡하고 dailyfeed-timeline, dailyfeed-content 등의 애플리케이션 측의 코드는 비교적 단순합니다. 프로젝트를 할때 가장 시간이 많이 들었던 요소는 install 스크립트 구성작업이었습니다. 나 혼자 하는 프로젝트이면 그냥 나만 알고 있는 방식으로 설치하면 되지만, 이 것을 실행시켜보려는 사람이 쉘스크립트 한번으로 모두 설치를 완료할수 있도록 하기 위해 작업을 하고, 중간에는 성능을 위해 MySQL, MongoDB, Redis, Kafka 가 사용하는 인프라를 분리한 install-hybrid.sh 라는 설치 파일을 하나 더 만들면서 설치 스크립트 구성 작업이 가장 힘들었던 요소 같습니다.
